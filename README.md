📰 Site_Sp_News

**São Paulo News – Portal de notícias sobre São Paulo e cidades da região, desenvolvido com HTML, CSS e JavaScript.
Projeto voltado à prática de desenvolvimento web, criação de conteúdo informativo e apresentação de notícias locais.**

📌 Sobre o Projeto

O SP News é um projeto criado para centralizar notícias e informações relevantes sobre regiões de São Paulo, facilitando o acesso da população a conteúdos locais de forma simples e organizada. Além de informar, o projeto demonstra minhas habilidades em desenvolvimento web, estruturação de conteúdo e criação de soluções digitais com foco no usuário.

O Site_Sp_News é um site tipo blog/notícias criado para exibir matérias, reportagens locais e informações relevantes sobre São Paulo e cidades vizinhas, como Taboão da Serra. Ele foi desenvolvido com tecnologias web básicas — HTML, CSS e JavaScript — com foco em estrutura semântica, layout visual agradável e responsividade leve.

O projeto funciona como um portal de notícias estático. Ele contém várias seções, incluindo manchetes, artigos sobre temas urbanos (como mobilidade, política municipal e fatos recentes), além de integração para links externos de fontes de informação.

🛠️ Funcionalidades

- ✔️ Cabeçalho com navegação e imagem representativa da cidade.
- ✔️ Seções de notícias organizadas por temas.
- ✔️ Artigos com estrutura clara e estilização agradável.
- ✔️ Uso de links externos para acompanhar andamento de obras e fontes oficiais.
- ✔️ Estilização com CSS customizado.
- ✔️ Componentes responsivos para boa leitura em diferentes telas.
